# OnlineCodeEditor

This project implements a program for a JS mentor and student. The program allows the mentor to follow up live on the student in a readonly while the student tackles a task from a given list.
\n The frontend is implemnted with react and the backend with node.js
\n The real-time follow up is implemnted with socket.io and the JS syntax highlting is done with the codemirror library
\n The project runs on a localhost
